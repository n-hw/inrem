# 개인정보처리방침

> **상태:** 초안 (Draft) · **법무 검토 전 — 법적 효력 없음**
> **발효일:** {{TBD}} · **버전:** 0.1 (개발용 베이스라인)

InRem(이하 "회사")은 「개인정보 보호법」(이하 "PIPA") 및 관련 법령에 따라 이용자의 개인정보를 보호하고, 관련 고충을 신속히 처리하기 위하여 본 처리방침을 둡니다.

---

## 1. 수집·이용 항목 및 목적

| 구분 | 항목 | 수집 시점 | 이용 목적 |
| --- | --- | --- | --- |
| 필수 | 이메일, 비밀번호(해시) | 회원가입 | 인증, 로그인 |
| 필수 | 디바이스 토큰 (FCM) | 앱 실행 | 푸시 알림(안부 체크) |
| 필수 | 활동 신호(시각, 신호 유형) | 앱 실행·활동 | Guardian Pulse 비활동 감지 |
| 선택 | 디지털 자산 정보(이름, 식별자, 메모, 암호화 비밀) | 사용자 입력 | Heritage Box 인벤토리 |
| 선택 | 보호자 정보 (이름, 연락처) | 사용자 입력 | 사용자가 지정한 보호자에게 비상시 알림 발송 |

**민감 정보(비밀번호·시드)**: 사용자가 Heritage Box에 입력한 민감 정보는 서버에 도달하기 전·후 모두 **Fernet 대칭 암호화**로 보관되며, 본인 외에는 복호화·열람할 수 없습니다 (rate-limit + 감사 로그 적용).

---

## 2. 보유 및 이용 기간

| 데이터 | 보유 기간 | 근거 |
| --- | --- | --- |
| 회원정보 | 회원 탈퇴 시까지 + **30일 grace** | PIPA 잊혀질 권리 |
| 활동 신호 | 최근 90일 | 서비스 운영상 필요 최소한 |
| 자산 인벤토리 | 회원 탈퇴 후 30일 grace → 영구 삭제 | 사용자 동의 |
| 푸시 토큰 | 회원 탈퇴 또는 디바이스 해제 시 | 즉시 |
| 감사 로그 (`inrem.audit.*`) | 1년 | 보안·법령 의무 |

탈퇴 후 30일 grace 기간 동안 **복구가 가능하며**, grace 종료 후에는 모든 개인정보가 영구 삭제됩니다.

---

## 3. 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 **제3자에게 제공하지 않습니다**. 다음의 경우는 예외입니다.

- 이용자가 사전에 동의한 경우 (예: 사용자가 지정한 **보호자**에게 비상 시 알림 발송).
- 법령에 의거하여 수사기관 등의 적법한 요청이 있는 경우.

## 5. 이용자 권리

이용자는 다음 권리를 행사할 수 있습니다.

1. **열람 요청**: 앱 내 "프로필" → 정보 확인.
2. **정정·삭제 요청**: 앱 내 각 항목별 수정·삭제 기능.
3. **처리 정지 요청**: 모니터링 OFF (설정 → 모니터링 활성화).
4. **회원 탈퇴 / 잊혀질 권리**: 앱 내 "설정 → 위험 영역 → 계정 삭제 요청" → 30일 grace 후 영구 삭제.

요청 처리 SLA: 영업일 기준 **10일 이내** ({{TBD: 회사가 정의}}).

## 6. 안전성 확보 조치

- **암호화**: 비밀번호 해시(bcrypt 등), 민감 자산 데이터 Fernet 암호화.
- **접근 통제**: JWT 기반 인증, 시크릿 reveal 엔드포인트 분당 10회 rate-limit + 감사 로그.
- **전송 구간 보호**: HTTPS/TLS 1.2+ 강제.
- **로그 관리**: 구조화 로그(`inrem.audit.*`)로 1년 보관, 접근 통제.

---

## 7. 자동 수집 장치

- **세션 토큰**: SecureStore에 저장, 30일 만료.
- **활동 신호(heartbeat)**: 사용자 동의 후 앱 실행/포어그라운드 시 자동 전송.
- **광고 식별자**: 사용하지 않음.
