## 1. 핵심 타겟 (Primary Target)

### 1순위: **1인 가구 중년 직장인 (40–55세)**
- **선택 이유**
  - "사회적 고립사"라는 명확한 페인 포인트가 존재 → 안부 체크 기능에 대한 즉각적 수요.
  - 디지털 친화도가 높아 모바일 앱 사용·구독 결제에 거부감이 적음 (구독 ARPU 확보 가능).
  - 부모 세대(60대 이상)에게 앱을 추천/대신 설정해줄 수 있는 "디지털 게이트키퍼" 역할.
  - Stage 1(안부) → Stage 2(엔딩노트·유산 인벤토리)로 자연스럽게 전환 가능한 라이프스테이지.

### 2순위: **부모 세대 자녀 (30–45세, 보호자 사용자)**
- 본인 사용보다 "부모에게 설치해 드린다"는 동기로 진입.
- 보호자(Guardian) 사이드의 알림·대시보드 경험이 핵심 가치.

### 3순위: **준비하는 시니어 (65–75세)**
- 디지털 격차가 있으나 자녀의 권유로 사용.
- UX 단순성·큰 글자·낮은 인지 부하가 핵심 성공 조건.

## 2. 페르소나 카드 (Persona Cards)

### 📇 페르소나 1 — 김재훈 (P1, 1차 페르소나)

| 항목 | 내용 |
| --- | --- |
| 한 줄 소개 | "혼자 사는 게 편하지만, 가끔 무섭다." |
| 나이/성별 | 47세 / 남성 |
| 직업 | IT 회사 차장 (재택 50%) |
| 거주 | 서울 마포구 오피스텔, 1인 가구 (8년차) |
| 가족 | 부모님(70대, 지방 거주), 미혼, 형제 1명 |
| 디지털 | iPhone 사용, 카카오·당근·토스·삼성헬스 능숙 |
| 소득 | 연 9,000만원 |

**핵심 니즈 (Needs)**
- 며칠 동안 연락 안 되어도 누군가는 알아주길 바람.
- 부모님께 자기 안위 걱정 끼치고 싶지 않음.
- 본인 사후 정리(계정·구독·가상자산)에 대해 막연한 불안.

**페인 포인트 (Pain Points)**
- 응급실 실려간 적이 있는데 회사·가족 모두 다음 날에야 알게 됨.
- 카카오 알림 같은 "지인 안부" 앱은 너무 가벼움 / 응급 SOS 앱은 너무 무거움.
- "엔딩노트 쓰자" 결심해도 어디서 시작할지 모름.

**우리 서비스 진입 트리거**
- 회사 동료의 1인 가구 돌연사 뉴스 → 검색 → 앱스토어 추천.

**성공 시나리오**
- 매일 무의식적으로 화면을 열기만 해도 안부 갱신.
- 8시간 무활동 → "잘 지내시나요?" 푸시 → 1탭으로 OK.
- 무응답 시 누나에게 자동 알림.
- 6개월 후 "엔딩노트 한 항목 작성하기" 프롬프트로 Stage 2 전환.

**거부 시나리오 (Anti-pattern)**
- 권한 요청이 과하거나(위치·연락처 등) "감시받는다"는 느낌이 들면 즉시 이탈.

### 📇 페르소나 2 — 박지영 (P2, 보호자 페르소나)

| 항목 | 내용 |
| --- | --- |
| 한 줄 소개 | "엄마가 혼자 계신 게 매일 마음에 걸린다." |
| 나이/성별 | 38세 / 여성 |
| 직업 | 마케팅 팀장 |
| 거주 | 경기 분당 (어머니는 부산 거주) |
| 가족 | 남편, 자녀 2명, 어머니(68세, 사별 후 5년) |
| 디지털 | iPhone, Instagram·Notion·Google Calendar 사용 |

**핵심 니즈**
- 어머니에게 부담 주지 않고 안전 확인.
- 이상 신호 시 "내가 가장 먼저" 알기.
- 효도 부담을 형제와 분담.

**페인 포인트**
- 매일 전화하지만 통화 못 한 날엔 죄책감.
- 어머니가 스마트워치는 거부 ("나 환자 취급 마라").

**진입 트리거**
- 맘카페에서 "혼자 사시는 부모님 안부 앱" 추천 글.

**성공 시나리오**
- 어머니 앱에 보호자 등록 → 24시간 무활동 알림 수신 → 전화 → 안심.
- Guardian 대시보드에서 평소 활동 패턴 확인.

### 📇 페르소나 3 — 이정숙 (P3, 시니어 사용자)

| 항목 | 내용 |
| --- | --- |
| 한 줄 소개 | "자식들 걱정시키고 싶지 않은 어머니." |
| 나이/성별 | 68세 / 여성 |
| 직업 | 은퇴 (전직 교사) |
| 거주 | 부산 해운대, 사별 후 1인 가구 |
| 디지털 | 카카오톡·유튜브 사용, 앱 설치는 자녀 도움 |
| 건강 | 고혈압·관절염 (관리 중) |

**핵심 니즈**
- 자식들에게 짐 안 되기.
- 죽음 이후 자식들이 헤매지 않도록 정리하기.

**페인 포인트**
- 작은 글씨·복잡한 메뉴는 못 씀.
- "사후 정리" 단어 자체가 부담스럽지만 필요성은 느낌.

**진입 트리거**
- 딸이 설치해 주고 사용법을 함께 익힘.

**성공 시나리오**
- 매일 아침 "오늘의 한 줄" 질문에 음성으로 답하기 (Life Narrator).
- 큰 버튼 하나로 안부 확인.
- 1년 후 자식이 엔딩노트 작성 권유 시 거부감 없음.

## 4. 안티-페르소나 (Anti-Persona, 의도적으로 제외)

- **20대 디지털 네이티브 1인 가구**: 죽음·돌봄 키워드에 거리감, 결제 전환률 낮음. (Stage 3 추모관 단계에서 재진입.)
- **요양원 입소자**: 시설이 모니터링을 대행, 앱 자체 사용성 부족.
- **고도 의료 모니터링 필요자**: 의료기기·간호 서비스 영역. 우리는 비-의료, 비-침습 영역에 집중.
